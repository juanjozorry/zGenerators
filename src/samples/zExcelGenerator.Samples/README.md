# zExcelGenerator Samples

This sample generates a simple Excel template with named ranges, fills it, and adds a new worksheet.

## Run

```bash
dotnet run --project src/samples/zExcelGenerator.Samples/zExcelGenerator.Samples.csproj
```

## Output

The sample writes two files into the current directory:

- `InvoiceTemplate.xlsx`
- `InvoiceOutput.xlsx`
